# 开始使用

1.在vue项目中的入口文件main.js导入样式

import 'jul-ui/dist/jul-ui.css'

2.引用

  • 所有组件均已封装在一起，直接在入口文件main.js导入
  • import 自定义名称 from 'jul-ui'
import JulUI from 'jul-ui'
  • 模块使用

同样在main.js里，使用JulUI组件 : Vue.use(自定义名称)

随后就可以开始使用

Vue.use(JulUI)
